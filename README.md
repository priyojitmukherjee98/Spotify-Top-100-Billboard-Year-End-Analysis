# Spotify Billboard Analysis

## 📌 Overview
This project analyzes Billboard chart data to identify trends in artist performance and ranking patterns.

## 🎯 Objective
To understand:
- Artist recurrence on charts
- Ranking distribution
- Performance consistency

## 🛠️ Tools Used
- SQL

## 🔍 Key Steps
- Queried and segmented chart data
- Analyzed ranking trends
- Evaluated artist consistency

## 📊 Key Insights
- Identified repeat-performing artists
- Observed ranking distribution patterns
- Discovered trends in chart performance

## 📈 Outcome
Provided data-driven insights into music trends and artist success patterns.
